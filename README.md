# AfriAnnonce Video Market

This is a minimal proof-of-concept for a video classifieds platform. Sellers can register and manage a store. An admin page allows changing the theme color. The site supports French, English and Arabic and works as a PWA.

## Setup

Install dependencies:

```bash
npm install
```

Run the development server:

```bash
npm start
```

The app will be available at `http://localhost:3000`.
